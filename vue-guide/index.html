Introduction

Overview

With this guide you'll be able to learn how to install vue and tailwindCSS with Vite, what Vue is and the basics of Vue. At the end we'll use vite with NodeJS to build a simple PLACEHOLDER app. Skip to the using Vite with NodeJS section if you already understand Vue and Vite.

What is Vue.js?

Vue (pronounced /vjuː/, like view) is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library is focused on the view layer only, and is easy to pick up and integrate with other libraries or existing projects. On the other hand, Vue is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries. (source)

What is Vite?

Vite (French word for "fast", pronounced /vit/) is a build tool that aims to provide a faster and leaner development experience for modern web projects. It consists of two major parts:

  • A dev server that provides rich feature enhancements over native ES modules, for example extremely fast Hot Module Replacement (HMR).
  • A build command that bundles your code with Rollup, pre-configured to output highly optimized static assets for production.

Vite is opinionated and comes with sensible defaults out of the box, but is also highly extensible via its Plugin API and JavaScript API with full typing support.

You can learn more about the rationale behind Vite in the Why Vite page. (source)

What is tailwindCSS?

TailwindCSS is a utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup. (source)

Prerequisites

Setup

This guide assumes you have the latest version of NodeJS and npm (or yarn) installed. Using a code editor like VScode is recommended.

Knowledge

This guide also assumes you have the following background:

  • Intermediate level knowledge of HTML, CSS, and JavaScript.
  • Basic understanding of NodeJS and express.
  • Optionally basic understanding of the terminal.

Installation And Setup

Install Vue with Vite

We can easily scaffold a Vue project with Vite with the following command, this will create a project folder named vue-app:

npm init vite@latest vue-app

Or if you're using yarn:

yarn create vite vue-app

You should see the following, use your arrow keys as stated in the cli to select Vue and press enter:

? Select a framework: › - Use arrow-keys. Return to submit. vanilla ❯ vue react preact lit-element svelte

Again, use your arrow keys to select vue and press enter:

✔ Select a framework: › vue ? Select a variant: › - Use arrow-keys. Return to submit. ❯ vue vue-ts

And with that we've scaffoled our project! Now we'll install the dependencies, run the following:

cd vue-app && npm install

Or with yarn:

cd vue-app && yarn install

And we're done installing Vue! Next we'll install tailwindCSS.

Install and setup tailwindCSS

Install tailwindCSS and its dependencies with the following command:

npm install -D tailwindcss@latest postcss@latest autoprefixer@latest

Or with yarn:

yarn add -D tailwindcss@latest postcss@latest autoprefixer@latest

Next, we'll create the config files:

npx tailwindcss init -p

Or with yarn:

yarn tailwindcss init -p

We'll edit our tailwind.config.js file and replace the "purge" field with the following:

purge: ['./index.html', './src/**/*.{vue,js}']

It should look like this:

module.exports = { purge: ['./index.html', './src/**/*.{vue,js}'], darkMode: false, // or 'media' or 'class' theme: { extend: {}, }, variants: { extend: {}, }, plugins: [], }

With that tailwindCSS can tree-shake unused styles in production builds by reading your code in the src directory/index.html file.

We're almost there! all we need to do now is import tailwindCSS in our app, for that we'll create a index.css file in src/assets folder and add the following in it:

@tailwind base; @tailwind components; @tailwind utilities;

And in src/main.js:

import './assets/index.css'

Your src/main.js file should look like this now:

import { createApp } from 'vue' import App from './App.vue' import './assets/index.css' createApp(App).mount('#app')

And we're done! Now start up the dev server by running:

npm run dev

Or with yarn:

yarn dev

If you're wondering why the Vue icon isn't centered or the counter button doesn't look like a default button - its because tailwindCSS removed their base styles to smooth over cross-browser inconsistencies.
